1、设置用户名：git config --global user.name "yuzhenbei"

2、设置邮箱：git config --global user.email "xiaoyu123@163.com" 

3、clone项目到本地 ：git clone git@github.com:yuzhenbei/TestAndroid.git

4、查看修改状态：git status

5、加入缓存区：git add 

6、提交到本地仓库：git commit -m "备注信息" 

7、代码提交到远程服务器：git push origin master
